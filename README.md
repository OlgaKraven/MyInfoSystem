# MyInfoSystem
Продукт должен решить несколько проблем, а именно фиксации задач и их статусов, обработки данных клиентов и их пожеланий, комментарии для сотрудников, чтобы они могли видеть взаимодействие между собой и клиентом. Как главных или основных пользователей проекта выделим две роли: менеджер и владелец.
